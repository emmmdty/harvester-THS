# 抖音内容投放审核筛选标准

版本：douyin-content-filter-v2026-05-25

## 目标

本标准用于 Step 1.5 抖音投稿内容投放前筛选，也可直接作为 Qwen 等模型的判断说明。判断口径是“抖音/巨量投放审核是否可过”，不是投后跑量效果。

默认运行配置：

- `FILTER_PROVIDER=qwen`：默认使用 Qwen 做内容初筛，兼容旧的 `STEP15_FILTER_PROVIDER`。
- `FILTER_STRICTNESS=balanced`：默认按批次风险排序放行约 40% 素材。
- `FILTER_TARGET_PASS_RATE=0.4`，`FILTER_MIN_PASS_RATE=0.33`，`FILTER_MAX_PASS_RATE=0.5`：通过量是软目标，硬拒绝和明确高风险内容不能为了凑比例而放行。

输出只允许 JSON：

```json
{
  "status": "pass|reject|review",
  "ruleIds": ["R1"],
  "briefReason": "一句话理由",
  "evidence": ["证据片段"]
}
```

- `pass`：未发现投放审核风险，可进入筛选后输出表。
- `reject`：明确命中硬拒绝规则，写为“不投放”。
- `review`：命中复核线索或证据不足，写为“需人工复核”。

## 输入字段

- `标题`：抖音标题或分享文案清洗后的标题。
- `tag词`：抖音话题标签。
- `文本`：页面正文、分享文案、标题、tag 的合并文本。
- `ASR`：视频语音转写文本，可能为空。
- `OCR`：视频帧或图文 OCR 文本，可能为空。
- `本地复核线索`：本地规则先扫出的 `review` 线索，模型必须结合上下文二次判断。
- `历史低过审先验`：历史反馈中低过审账号或内容类型，仅作为复核线索，不直接硬拒绝。

## 判断原则

1. 不要因为单个金融词直接拒绝。仅出现“股票、投资、基金、涨停、复盘、加仓、抄底、必看”等词时，先看它是在客观资讯、段子、书单、历史回顾，还是在诱导用户交易。
2. 硬拒绝只用于明确违规：具体荐股、股票代码、目标价、收益承诺、保本无风险、明确买卖或仓位指令、高危金融业务。
3. 复核线索用于降低漏判：资金流、主力净流入、选股模型、涨停复盘、产业链、财富曲线、理财书单等样本通过率波动大，默认 `review`，除非上下文明确安全或明确违规。
4. 同一素材同时命中硬拒绝和复核线索时，以硬拒绝为准。
5. 证据必须引用标题、tag、ASR、OCR 或文本中的短片段，不要编造未出现的信息。
6. 批次放行按风险从低到高排序，默认约 1/3-1/2 进入初筛通过池；安全候选不足时不强行补足。
7. 资讯内容更严格，非资讯内容更宽松：`资讯` 命中复核线索时优先保留复核；图文、盘点、股友说等非资讯轻度复核内容可按批次风险排序进入通过池。
8. 历史回放评估必须保护正例：历史 `是否投放成功=是` 且未命中硬拒绝的内容保持 `通过`，但生产新内容不得使用历史标签作弊。

## 档位与批次配额

- `strict`：目标通过率约 33%，只放行模型明确 `pass` 且无明显复核线索的低风险素材。
- `balanced`：默认，目标通过率约 40%，优先放行低风险素材，少量轻度 `review` 可按排序进入通过池。
- `loose`：目标通过率约 50%，扩大轻度复核内容的放行范围，但硬拒绝和模型明确 `reject` 永远不放行。

最终写回规则：

- 进入通过池：写 `通过`，进入 `抖音筛选结果`。
- 未进入通过池但非硬拒绝：写 `需人工复核`。
- 命中硬拒绝或模型明确高风险：写 `不投放`。

账号历史通过率作为评估基线：

| 账号 | 当前通过率 | 策略 |
| --- | ---: | --- |
| 投资号 | 50% | 相对宽松，但通过池历史过审率需高于 50%。 |
| 股民社区 | 51% | 相对宽松，但通过池历史过审率需高于 51%。 |
| 财经号 | 30% | 严格控制，低风险内容优先，边界内容复核。 |
| 问财 | 31% | 严格控制，尤其资金流/选股模型类不默认放行。 |
| 理财 | 42% | 中等偏严格，财富焦虑/收益暗示复核。 |
| 期货通 | 47% | 中等偏严格，期货交易导向内容复核。 |

## 规则清单

### 硬拒绝

- `R1 股票代码/具体荐股`：出现股票代码、具体标的推荐、目标价、个股推荐。例：`600519 今天买点`、`这只股目标价 30`。
- `R2 收益承诺/盈利诱导`：出现稳赚、保本、无风险、包赚、必涨必跌、跟着操作就赚钱。例：`跟着买稳赚`、`收益翻倍`。
- `R3 直接买卖/仓位指令`：出现立即买入、建议加仓、可以抄底、买点来了、止盈止损等明确操作建议。
- `R4 投资风险提示缺失`：直接销售或推广股票、基金、期货、理财产品、投顾/荐股服务，但没有显著风险提示。
- `R5 高危金融/非法金融业务`：股票配资、场外配资、私募、信托、P2P、校园贷、二元期权、虚拟货币、荐股软件、内幕消息等。

### 复核线索

- `R6 绝对化用语`：最强、最好、最高、唯一、首选、第一、顶级等，需要判断是否为广告夸大。
- `R7 难以证实的数据`：用户数、投资者规模、行业公认等缺少来源的数据。
- `R8 不文明用语`：接盘侠、韭菜、割韭菜、垃圾、骗子等可能冒犯用户或引发负面情绪的表达。
- `R9 行情复盘/涨停/点位`：涨停复盘、涨停潮、指数点位、机构加仓新闻等。客观资讯不直接拒绝，但需确认没有交易建议。
- `R10 资金流/选股模型`：主力净流入、大单净买入、资金流向、龙虎榜、选股模型、领涨个股、龙头强度等。
- `R11 产业链/概念/名人财富`：产业链、概念股、供应商、龙头、财富曲线、身家故事等，需要判断是否导向具体投资。
- `R12 理财/财富自由/投资书单`：理财、财富自由、复利、搞钱、存钱、必看书籍等，需要判断是否承诺收益或制造焦虑。

## 样本边界

来自 `原生内容投稿 (1).xlsx` 的真实抖音投放反馈显示，以下内容不应被本地规则直接硬拒绝：

- `5月18日涨停股复盘`：命中 `R9`，默认复核；若只是客观复盘且无买卖建议，可 `pass`。
- `期货必看书籍推荐`：命中 `R12` 或 `R6`，默认复核；若只是书单推荐且无收益承诺，可 `pass`。
- `全国社保基金多次在A股大幅下跌时果断加仓`：命中 `R9`，默认复核；若为新闻叙述，可 `pass`。

以下内容应进入 `review` 或 `reject`：

- `主力资金净流入锁定行业，大单净买入超1亿，输出领涨个股`：命中 `R10`，至少 `review`；若推荐具体买入标的，`reject`。
- `存100万要多久`、`财富自由三大铁律`：命中 `R12`，至少 `review`。
- `600519 今天出现买点，目标价 30，跟着买稳赚`：命中 `R1/R2/R3`，`reject`。
